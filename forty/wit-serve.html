wit-serve

Main server for all wit products.

  • Get the code

Introduction

A nodejs application running the nest.js framework for building efficient and scalable server side applications.

Provides licensing, authentication and federation to other wit services. Licensing, authentication, caching et al are provided through modules.

Showcasing this project is:

  • MongoDB - document (JSON) storage for all licenses and accounts.
  • vscode - my goto editor for web development, from the team @ Microsoft.
Dependencies:
  • node.js - minimum of V8.x.x, which includes npm
  • git and a github account

Creating the Project

Step Description
1 Install nest.js cli (scaffholding)
npm install -g @nestjs/cli
2 Create the app
nest new wit-serve
3 Build the app
cd wit-serve
npm install
4 Run the app
npm start
5 Test the app, by opening a browser at:
http://localhost:3000
Expect to see "Hello World!". <CTRL><C> to quit.
6 Initialise as a new git repo and add the remote github repo:
git init
git remote add origin https://github.com/wozitech/wit-serve.git
7 Create a .gitignore file with node_modules:
cat .gitignore
# Node modules
/node_modules
8 Add, commit files back to github (set-upstream option only required on first push):
git add --all
git commit -m "Initial Commit"
git push --set-upstream origin master
